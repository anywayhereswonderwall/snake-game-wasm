### Installation

1. Install Rust
   ```sh
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
2. Install wasm-pack
   ```sh
   curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
   ```
3. Install NPM
   ```sh
   npm install npm@latest -g
   ```
